Bubble Sort

Bubble Sort repeatedly swaps adjacent elements if they are in wrong order until array is sorted.


// C++ example

Linear Search

Checks each element sequentially to find a target value in array.


// C++ example
int linearSearch(int arr[], int n, int x) {
  for (int i = 0; i < n; i++)
    if (arr[i] == x)
      return i;
  return -1;
}
